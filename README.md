# ✨ LuminaWrite

> **AI-Powered Writing Assistant | Grammarly Lite Built with React & Gemini AI**

LuminaWrite is a modern AI-powersed writing assistant that helps users improve their writing by providing real-time grammar, spelling, punctuation, fluency, and style corrections. Inspired by Grammarly, the application offers an intuitive interface where users can review and apply AI-generated suggestions individually or all at once.

---

## 🚀 Features

- ✍️ AI-powered grammar and spelling correction
- 📝 Punctuation and sentence structure improvements
- 🎯 Multiple writing tones
  - Professional
  - Academic
  - Casual
  - Creative

- 💡 Interactive suggestion panel
- ✅ Apply individual or all suggestions
- ⚡ Fast response using Google Gemini 2.5 Flash
- 📱 Fully responsive UI
- 🌙 Modern dark theme with glassmorphism design
- 🎨 Smooth animations and clean user experience

---

## 🛠️ Tech Stack

### Frontend

- Google Gemini API
- Gemini 2.5 Flash

### Styling

- Vanilla CSS

---

## 🎯 How It Works

1. Enter or paste your text.
2. Select your preferred writing tone.
3. Click **Correct Grammar**.
4. Review AI-generated suggestions.
5. Apply changes individually or all at once.
6. Copy your polished content.
