✨ LuminaWrite

AI-Powered Writing Assistant | Grammarly Lite Built with React & Gemini AI

LuminaWrite is a modern AI-powered writing assistant that helps users improve their writing by providing real-time grammar, spelling, punctuation, fluency, and style corrections. Inspired by Grammarly, the application offers an intuitive interface where users can review and apply AI-generated suggestions individually or all at once.

🚀 Features
✍️ AI-powered grammar and spelling correction
📝 Punctuation and sentence structure improvements
🎯 Multiple writing tones
    Professional
    Academic
    Casual
    Creative
💡 Interactive suggestion panel
✅ Apply individual or all suggestions
⚡ Fast response using Google Gemini 2.5 Flash
📱 Fully responsive UI
🌙 Modern dark theme with glassmorphism design
🎨 Smooth animations and clean user experience

🎯 How It Works
    Enter or paste your text.
    Select your preferred writing tone.
    Click Correct Grammar.
    Review AI-generated suggestions.
    Apply changes individually or all at once.
    Copy your polished content.
